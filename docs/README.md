# Website

This website is built using [VitePress](https://vitepress.dev).

## Installation

```bash
npm install
```

## Local Development

```bash
npm run dev
```

This command starts a local development server. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `.vitepress/dist` directory and can be served using any static content hosting service.

## Preview

```bash
npm run preview
```

Previews the production build locally.
